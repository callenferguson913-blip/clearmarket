# ClearMarket

A weekly AI-powered investment report service for beginner investors. Live at [clearmarket.live](https://clearmarket.live).

## What it does

Every Monday at 8am Eastern, ClearMarket emails each user a personalized investment report built around their profile — goal, risk tolerance, weekly investment amount, and current holdings. Reports are generated by Claude (Anthropic) using live market data and news pulled from Yahoo Finance.

Users can update their preferences at any time via a magic link included in every email.

## Stack

- **Backend:** Python, FastAPI
- **Database:** PostgreSQL (Railway), SQLAlchemy ORM
- **AI:** Anthropic Claude API (`claude-sonnet-4-6`)
- **Market data:** yfinance (live prices + news headlines)
- **Email:** iCloud SMTP
- **Hosting:** Railway
- **Automation:** GitHub Actions (cron, every Monday 8am Eastern)

## Project structure

```
clearmarket/
├── main.py              # FastAPI app — routes for signup, settings, homepage
├── send_reports.py      # Weekly job — fetches data, generates reports, sends emails
├── start.py             # Uvicorn entry point (reads $PORT from Railway)
├── src/
│   └── database.py      # SQLAlchemy models: User, Recommendation, MagicToken
├── templates/           # Jinja2 HTML templates
├── static/              # CSS
├── Dockerfile           # Used to bypass Railway's Nixpacks scanner
└── .github/workflows/
    └── weekly_reports.yml  # GitHub Actions cron job
```

## Key features

- Personalized reports per user profile (goal, risk, interests, holdings)
- Live market data fetched at send time — prices always current
- Recommendation tracking: tickers saved weekly with price at recommendation
- Performance tracking: percent change calculated the following week
- Homepage displays top performing picks automatically
- Magic link in every email lets users update preferences without a login
- Retry logic for Anthropic API overload errors

## Environment variables

| Variable | Description |
|---|---|
| `DATABASE_URL` | PostgreSQL connection string |
| `ANTHROPIC_API_KEY` | Anthropic API key |
| `EMAIL_ADDRESS` | iCloud email address for sending |
| `EMAIL_APP_PASSWORD` | iCloud app-specific password |
| `BASE_URL` | Production URL (https://clearmarket.live) |
| `TEST_MODE` | Set to `true` to only send to the owner email |

## Running locally

```bash
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
cp .env.example .env  # fill in your values
python start.py
```
